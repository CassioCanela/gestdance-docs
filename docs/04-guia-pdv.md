# GUIA DO USUÁRIO (PDV)

## O que é o PDV
Use o módulo de PDV para registrar vendas no balcão e acompanhar comissões relacionadas.

## Rotas principais
- PDV (Balcão): `/pdv`
- PDV Rápido: `/quick-pos`

## Passos básicos
1. Abra o PDV.
2. Selecione produtos e registre a venda.
3. Finalize o pagamento conforme o método escolhido.

## Onde ver comissões e relatórios
- Consulte as páginas do módulo de vendas/comissões e relatórios vinculados às operações do PDV.
