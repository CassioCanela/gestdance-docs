# Como Gerar Recibos

## Recibos Automáticos
Recibos são gerados automaticamente quando uma venda é registrada.

## Acessar Recibos
1. Clique em "Recibos"
2. Filtre por: Data, Aluno, Status (pago, pendente)
3. Clique em um recibo para visualizar

[Screenshot: lista de recibos]

## Visualizar Recibo
1. Clique no recibo
2. Revise os dados: Aluno, Produtos, Valor total, Data, Forma de pagamento
3. Clique em "Baixar PDF" para salvar

[Screenshot: visualização de recibo]

## Reenviar por Email
1. Clique no recibo
2. Clique em "Reenviar por Email"
3. Confirme o email do aluno
4. Clique em "Enviar"

[Screenshot: reenvio de email]

## Imprimir
1. Clique no recibo
2. Clique em "Imprimir"
3. Selecione impressora
4. Clique em "Imprimir"
